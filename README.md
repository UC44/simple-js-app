# Pokédex App
To view a picture and stats, search for your preferred Pokémon. Pokémon API. is employed by the app to load a list of Pokémon. Javascript is used to fill up the list.
## Technologies Used
- HTML
 - CSS
 - JavaScript
 - jQuery
 - Bootstrap 4.3.1
